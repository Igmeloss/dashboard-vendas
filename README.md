# Projeto: Dashboard de Vendas e Lucratividade - Power BI

Este projeto consiste no desenvolvimento de um dashboard interativo no **Power BI Desktop**, focado no monitoramento visual dos principais KPIs de vendas de uma empresa fictícia. O objetivo foi criar uma visualização intuitiva e moderna (com design Dark Mode) que permitisse uma análise rápida da performance comercial.

## 🎯 O que este dashboard analisa

Este painel foi estruturado para responder a perguntas de negócio essenciais:

1.  **Visão Geral (KPIs):** Apresentação instantânea do Faturamento Total, Custo, Lucro e Margem de Lucro (em porcentagem).
2.  **Sazonalidade:** Gráfico de Receita ao longo do tempo (mensal) para identificar tendências de vendas.
3.  **Análise de Desempenho:**
    * Ranking dos principais vendedores (Top Vendedores).
    * Detalhamento de performance por categoria de produto (ex: Eletrônicos, Brinquedos, Móveis).
    * Ranking dos produtos mais vendidos.
4.  **Filtros Interativos:** Possibilidade de segmentar os dados por Calendário (Período), Região, Categoria, Produto e Vendedor.

## 🛠️ Tecnologias Utilizadas

* **Power BI Desktop:** Utilizado para a importação de dados, modelagem, criação dos cartões de KPI e design dos gráficos.

## 🚀 Próximos Passos (Melhorias Futuras)

Como o foco deste projeto foi o desenvolvimento visual (front-end no Power BI), as próximas etapas poderiam incluir:
* Integração com uma base de dados mais complexa via SQL.
* Uso de linguagem DAX para métricas e medidas mais complexas.
* Criação de tooltips customizados para os gráficos de barras.
